# Examples

Example scripts in `docs/examples/`.

- [basic_usage.py](basic_usage.py)
- [websocket_monitoring.py](websocket_monitoring.py)
- [ingest_raw.py](ingest_raw.py)
- [query_gulp_preview.py](query_gulp_preview.py)
- [collab_notes_links.py](collab_notes_links.py)
- [storage_list_files.py](storage_list_files.py)
- [query_external_elasticsearch.py](query_external_elasticsearch.py)

These files are Python scripts and can be run directly (require a local gULP server running on `localhost:8080` or adjust `GULP_BASE_URL` env var). 

They demonstrate common workflows such as authentication, operation management, document ingestion, querying, collaboration features, and storage management.
